# Designo

### Figma-Style Design Editor using HTML, CSS & JavaScript (DOM)

Designo is a **Figma-style visual design editor** built using **pure HTML, CSS, and JavaScript (DOM)**.
The project focuses on understanding how modern design tools work internally by recreating their core features without using any external libraries, frameworks, Canvas API, SVG, or images.
